1. Start MySQL Installer, should be in your start menu under MySQL folder.
  2. if you can see workbench in window opened you don't have to follow the remaining steps.
  3. click add near top right of window.
    1. expand MySQL applications all the way and find workbench.
    2. select it and click on arrow point right, it should be on right side of screen now.
    3. click next and finish the installation.
