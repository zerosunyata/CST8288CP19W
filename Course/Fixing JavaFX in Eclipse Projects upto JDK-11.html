Do not use JDK 11. use 10, 9 or 8.

Change JRE to JDK in eclipse

open eclipse and navigate to window/preferences.
in the left menu find java/installed JREs.
select whatever installed JRE with a tick beside it and select edit.
in new window click Directory and navigate to "C:\Program Files\Java\".
in second line change the name from jre[version] to jdk[version] so you know.
find the newest version of JDK (NOT JRE) select it and click Select Folder.
click finish and then apply and close.

Change your project JRE to Workspace JRE

right click on your project and select Build Path/Configure Build Path.
in the left menu select Java Build Path.
select Libraries from middle top.
select JRE System Library and click Edit.
in new window select Workspace default JRE. it should say jdk[version] in front of it.
click Finish and Apply and Close.
