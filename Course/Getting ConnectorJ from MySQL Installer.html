1. Start MySQL Installer, should be in your start menu under MySQL folder.
  2. if you can see Connector/J in opened window you don't have to follow the remaining steps.
  3. click add near top right of window.
    1. expand MySQL Connector and find Connector/J. expand all the way.
    2. select it and click on arrow pointing right, it should be on right side of screen now.
    3. click next and finish the installation.
  4. jar file you need is located in: C:\Program Files (x86)\MySQL\Connector.J 5.1, version might be 8.x as well.
