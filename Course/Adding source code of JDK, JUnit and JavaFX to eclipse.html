1. open eclipse and navigate to "help"/"install new software".
  2. from the drop down menu select "Oxygen - http://download.eclipse.org/releases/oxygen" or "Photon - http://download.eclipse.org/releases/photon" if not there paste it in.
  3. wait for it to load and under "Application Development Frameworks" select "Eclipse Project SDK".
  4. click next and continue to install it.
  5. after a restart you will have access to all source codes for JDK, JUnit, JavaFX and eclipse components.
